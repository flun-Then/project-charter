# Aplikasi Pengelolaan Data Mata Kuliah Berbasis Desktop

## Deskripsi Proyek

Aplikasi Pengelolaan Data Mata Kuliah Berbasis Desktop merupakan aplikasi yang dirancang untuk membantu pengguna dalam mengelola data mata kuliah secara lebih terstruktur dan efisien. Aplikasi ini memungkinkan pengguna untuk menambahkan, mengubah, menghapus, serta menampilkan informasi mata kuliah seperti kode mata kuliah, nama mata kuliah, dan dosen pengampu.

Proyek ini dikembangkan sebagai bagian dari kegiatan pembelajaran pengembangan perangkat lunak dengan durasi pengerjaan selama 8 minggu. Tujuan dari pengembangan aplikasi ini adalah untuk mempermudah pengelolaan data akademik dengan memanfaatkan teknologi aplikasi desktop.

Dengan adanya aplikasi ini, pengelolaan data mata kuliah dapat dilakukan secara lebih sistematis dibandingkan pencatatan manual, sehingga dapat meningkatkan efisiensi dan meminimalkan kesalahan dalam pengolahan data.

## Fitur Utama

* Menambahkan data mata kuliah
* Mengubah data mata kuliah
* Menghapus data mata kuliah
* Menampilkan daftar data mata kuliah
* Menyimpan data secara terstruktur

## Teknologi yang Digunakan

* Bahasa Pemrograman: Java
* GUI Framework: Java Swing
* IDE: NetBeans
* Database: MySQL (opsional jika menggunakan database)

## Cara Menjalankan Aplikasi

1. Clone repository ini ke komputer
2. Buka project menggunakan NetBeans atau IDE Java lainnya
3. Compile dan jalankan program
4. Gunakan aplikasi untuk mengelola data mata kuliah
